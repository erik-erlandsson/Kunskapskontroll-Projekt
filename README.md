# Kunskapskontroll-Projekt

## Plankinspektion

A project for inspecting and managing fence/plank structures.

## About

This is a knowledge control project (Kunskapskontroll) focused on plank inspection systems. The project aims to provide tools and documentation for systematic inspection of fence and plank structures.
